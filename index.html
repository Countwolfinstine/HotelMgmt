HOW DOES IT WORK?

It All Starts With Data

The student marks data including the relevent course information is uploded in the prescribed format through the upload page. The platform preprocesses the data and then the mean, average, variance, etc. is calculated. The processed data is saved in the database for furthur uses.

Visualize The data From Every Angle

The platfrom creates a staistical metric using the processed data. The platform uses this metric to figure out the best possible way to visualize the processed data. The visualitaion is done and presented to the user on the dashboard.

The Moderation

The platfrom has extensive tools to visualize and moderate the students data according to specific requirenments. The platform enables the user to make specific changes to the data set and see the changes in the form of updated visualizations in real-time. The changes made to the data can be saved while retainig the original data

Today's Menu

On the back-end, the Modratr platform comprises a suite of capabilities for integrating tabular data into coherent models, performing complex computations over models, and sharing and iterating on analytic products.

DATA STORAGE

The platform uses multiple databases to store the uploded data. The data set is tracked and periodically saved and all changes are logged. All the copies of the modified data is also saved

VISUALIZATION ENGINE

The platfrom has powerful and flexible visulization capabilities. It uses several different visualization techniques to represent the data according to the need of the user.

MODERATION ENGINE

Using the vast moderation faclities provided by the platform, the user can moderate the data according to the specifications. It tools which suggests the users on how to best moderate the data.

HISTORY SENSITIVE

The platform saves the changes made to the dataset in real time so that the user doesnt loose their changes in case of any faliures. The user can also go back to any version of the changed dataset.

ANOMOLY DETECTION

The Platform has the ability to detect and correct any anomolies in the uploaded data. It also alerts the user about the anomolies and the corrective action it has taken to resolve the said anomolies

DATA ARCHIVING

Every data uploded to the platform is saved in the database and any moderation done by the user is refelected on a copy of the data. The original is not touched.
